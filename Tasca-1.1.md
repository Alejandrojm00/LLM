# Headers
Para poner texto con diferentes headers se utiliza```"#" o "h1, h2..."```cuantas más se utilizan mas pequeño será el texto.
Diferentes ejemplos serían: 
1. ## Ej1.
2. ### Ej2.
3. #### Ej4.

 # Emphasis
 Para poner enfasis en un texto utilizaremos:
 - Para negrita ```"*"```.
 - Para cursiva ```"_"```.
 - Para tachado ```"~~"```.
